Privacy Policy — Structune

Last updated: June 2026

Structune does not collect any personally identifiable information. The extension sends your builder input to Google's Gemini API when you click Generate, and separately sends anonymous, aggregated feature-usage events (e.g. which button was clicked) to Google Analytics (GA4) to help us understand which features are used. No prompt content, API keys, or saved Library data are ever included in analytics events.

What the Extension Does

  • Sends your builder input to the Google Gemini API using the API key you provide, when you click Generate.
  • Displays the structured spec result locally in the popup.
  • Stores your saved prompts, settings, and draft locally in your browser.
  • All processing happens on your device. Structune does not operate any backend server.

Google Gemini API

When you click Generate, the text you entered in the builder is sent to Google's Gemini API (generativelanguage.googleapis.com) using your own API key. This request is made directly from your browser to Google's servers — no intermediate server is involved.

Google's privacy policy governs how your data is handled by the Gemini API: https://policies.google.com/privacy

Google Analytics (GA4)

Structune uses Google Analytics 4 to collect anonymous, aggregated statistics about which features are used (e.g. Generate, Save, Library filters, Settings). Each event is tied only to a randomly generated client ID stored on your device — never to your name, email, Gemini API key, prompt content, or saved Library data.

Google's privacy policy governs how Google Analytics handles this data: https://policies.google.com/privacy

What the Extension Does Not Do

  • Does not collect or transmit your API key to any server other than Google's Gemini API.
  • Does not track browsing history or access any website content.
  • Does not include prompt content, API keys, or Library data in analytics events.
  • Does not use crash reporting or any third-party SDK other than Google Analytics.
  • Does not sync data across devices.
  • Does not execute remotely hosted code.
  • Does not require an account of any kind.
